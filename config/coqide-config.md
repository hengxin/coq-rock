## Shortcuts

- `Ctrl + Downarrow`: Forward one command
- `Ctrl + Uparrow`: Backward one command
- `Ctrl + Home`: Backward to the first command (Restart Coq)
- `Ctrl + End`: Forward to the end command (Go to end)
- `Ctrl + `: Go to the cursor
